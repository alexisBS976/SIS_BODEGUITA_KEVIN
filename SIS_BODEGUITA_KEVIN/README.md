Sistema de Gestión de Bodeguita Kevin

Descripción
Sistema de gestión de bodega desarrollado en C# con Windows Forms. Permite administrar productos, registrar ventas, controlar el inventario y generar reportes.

Requisitos

- Visual Studio 2022 o superior.
- .NET 10.0 Desktop Runtime.
- Windows 10 o Windows 11.

Dependencias

> - Microsoft.Data.Sqlite
> - Archivo "BD_BodeguitaKevin.db"
> - No requiere instalación de SQL Server.

Cómo ejecutar el proyecto

1. Clonar  el repositorio.
2. Abrir el archivo "SIS_BODEGUITA_KEVIN.sln" en Visual Studio.
3. Copiar y reemplazar la base de datos hacia la ruta: SIS_BODEGUITA_KEVIN\bin\Debug\net10.0-windows
4. Compilar la solución.
5. Ejecutar el proyecto con **F5**.

Credenciales de acceso

Usuario: `admin`

Contraseña: `1234`

Funcionalidades

- Inicio de sesión.
- Registro de ventas.
- Consulta de inventario.
- Actualización de stock.
- Reportes de productos, ventas y detalle de ventas.
